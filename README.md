# Proyecto E-commerce: Dietética

## Propósito del Proyecto

Este proyecto es un E-commerce de Dietética desarrollado como parte del curso de talento tech, integrando múltiples funcionalidades interactivas y buenas prácticas de desarrollo web.

## Características Implementadas

* **Estructura Semántica:** Uso de HTML5 (`<header>`, `<main>`, `<section>`, etc.) y principios básicos de SEO.
* **Diseño Responsivo:** Layout adaptable a diferentes pantallas utilizando:
    * **Flexbox:** Para la organización de los botones de categoría y las tarjetas de productos.
    * **CSS Grid:** Para la sección de reseñas de clientes.
    * **Media Queries:** Para la adaptación general en dispositivos móviles.
* **Interactividad Avanzada (JavaScript):**
    * **Filtro Dinámico:** Filtrado de productos por categoría usando el DOM.
    * **Consumo de API:** Uso de `Fetch API` (FakeStoreAPI) para obtener datos de productos (título, precio).
    * **Validación de Formulario:** Validación básica de campos requeridos y formato de email.
* **Carrito de Compras:**
    * **Persistencia de Datos:** El estado del carrito se guarda en `localStorage`.
    * **Funcionalidad Completa:** Permite agregar productos, actualizar cantidades, eliminar ítems y muestra el total de la compra dinámicamente.
* **Accesibilidad (A11y):** Uso de atributos `alt` en imágenes y enfoque navegable por teclado.
